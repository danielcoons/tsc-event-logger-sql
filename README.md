# tsc-event-logger-sql
An extension to the TSC Event Logger to write to a SQL Database

Find the VIP here: [![Image](https://www.vipm.io/package/tsc_lib_tsc_event_logger_sql/badge.svg?metric=installs)](https://www.vipm.io/package/tsc_lib_tsc_event_logger_sql/) [![Image](https://www.vipm.io/package/tsc_lib_tsc_event_logger_sql/badge.svg?metric=stars)](https://www.vipm.io/package/tsc_lib_tsc_event_logger_sql/)

Extend the [TSC Event Logger](https://github.com/danielcoons/tsc-event-logger) by adding a SQLite Database output.

Follows Structured Logging principles for LabVIEW Application event logging.
